# Changelog

All notable changes to Viralyn will be documented in this file.

## [1.0.0] - 2026-02-26

### Added
- Initial release of Viralyn platform
- User authentication (JWT-based)
- Campaign management for brands
- Influencer dashboard
- Campaign assignment workflow
- Deliverable submission and verification
- Payment tracking system
- Analytics dashboard
- Responsive design for mobile and desktop
- Real-time notifications with toast system
- Loading states and skeleton loaders
- Premium landing page
- Supabase PostgreSQL integration
- Comprehensive documentation

### Features

#### For Brands
- Create and manage campaigns
- Assign influencers to campaigns
- Verify deliverables
- Track payments
- View analytics and metrics
- Manage influencer database

#### For Influencers
- View assigned campaigns
- Accept/decline campaign invitations
- Submit deliverables with metrics
- Track payment status
- View campaign history

### Technical
- Frontend: Next.js 14 with App Router
- Backend: NestJS with TypeScript
- Database: PostgreSQL (Supabase)
- ORM: Prisma
- UI: Tailwind CSS + Shadcn/ui
- Authentication: JWT
- Testing: Jest + React Testing Library

### Documentation
- Setup guides
- API documentation
- Database schema
- Deployment guides
- Contributing guidelines
